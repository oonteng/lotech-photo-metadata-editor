# Contributing

LOTECH Photo Metadata Editor is currently maintained as a focused macOS utility.

Before changing behavior, keep the existing philosophy intact:

- The user edits logical metadata fields, not EXIF/IPTC/TIFF implementation details.
- Batch Edit remains a spreadsheet-style metadata workspace.
- Single Edit remains the canonical workflow for detailed photo correction.
- Saving should be explicit, reversible where practical, and transparent when it fails.

Run a clean Xcode build before submitting changes.
