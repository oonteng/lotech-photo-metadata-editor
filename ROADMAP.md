# Roadmap

## Version 1.0

- Production-ready Single Edit.
- Production-ready Batch Edit.
- Reliable metadata save workflow for common photo formats supported by ImageIO.
- Documented LOTECH logical metadata model.

## Future

- Dedicated XMP packet writer or vetted metadata engine integration.
- Broader RAW metadata support.
- Sidecar metadata support for formats that should not be modified directly.
- More batch validation tools for large archive cleanup.
- Shared LOTECH Metadata Engine for future Photo, Video, Audio, and Document editors.
